# Plastic-calculator

Calculadora de Consumo de Plástico

Projeto desenvolvido como atividade de extensão universitária com o objetivo de promover a conscientização ambiental sobre o consumo de plástico descartável.

Funcionalidades

- Cálculo estimado de consumo de plástico semanal, mensal e anual
- Estimativa do peso de plástico descartado
- Equivalência aproximada em garrafas PET
- Simulação de redução de consumo
- Impacto coletivo de redução de plástico
- Visualização gráfica dos dados

Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Chart.js
- GitHub Pages

Objetivo do projeto

A ferramenta foi desenvolvida para auxiliar na conscientização ambiental, permitindo que os usuários estimem o impacto do consumo de plástico no dia a dia e reflitam sobre possíveis mudanças de comportamento.
